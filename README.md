# 챱츄르냥이 🐱

2레인 리듬 게임. Phaser 3 기반, 단일 HTML 파일로 동작합니다.
모바일/데스크톱 모두 지원합니다.

## 플레이 방법

- **데스크톱**: `F` 또는 `←` (왼쪽 레인), `J` 또는 `→` (오른쪽 레인)
- **모바일**: 화면 좌/우 영역을 탭
- 노트가 판정선에 도달하는 순간 해당 레인을 누르세요.
- **홀드 노트**(긴 막대)는 머리를 누른 채 끝까지 유지하다가 꼬리에 맞춰 떼야 합니다.
- 누르고 있는 동안에는 효과음이 연속으로 재생됩니다.

## 채점 / 콤보

- 판정: `완벽!` / `굿!` / `좋아` / `실패`
- **RPM** = 맞춘 블록 수 × (100 + 콤보 × 25)
- **콤보 5** 이상 달성 시 BGM 재생 시작
- **콤보 10** 이상부터 화면 중앙에 임팩트 콤보 숫자 표시 (콤보가 깨지면 초기화)
- **콤보 30, 40, 50…** 10의 배수마다 `miao.mp3` 효과음 + 캐릭터 주변 하트 연출
- 곡 종료 시 정확도에 따라 등급 부여 (S / A / B / C / D)

## 곡 / 패턴

- 길이: 60초, BPM 100
- 매 플레이마다 시드 기반으로 패턴이 새로 생성됨
- 구간별 난이도: intro → verse → chorus → build → outro

## 에셋

| 파일 | 용도 |
| --- | --- |
| `1.png` / `2.png` / `3.png` | 하단 캐릭터 (idle / left press / right press) |
| `fish.png` | 노트 이미지 |
| `heart.png` | 콤보 임팩트 시 캐릭터 주변에 흩날리는 하트 |
| `chap.mp3` / `chu.mp3` | 레인 입력 효과음 (좌/우) |
| `back.mp3` | 배경 BGM (콤보 5 이상에서 재생) |
| `miao.mp3` | 콤보 30/40/50… 임팩트 효과음 |

## 실행

별도 빌드/의존성 없음. 정적 파일 서버로 띄우면 됩니다.

```bash
# 예: 파이썬 내장 서버
python3 -m http.server 8000
# http://localhost:8000 접속
```

> 주의: `file://` 직접 열기는 일부 브라우저에서 오디오/이미지 로딩이 차단될 수 있으므로 HTTP 서버로 여는 것을 권장합니다.

## 기술

- [Phaser 3.60](https://phaser.io/) (CDN)
- WebAudio API로 효과음/BGM 저지연 재생 (iOS Safari 언락 처리 포함)
- 캔버스 스케일 모드: `FIT` + `CENTER_BOTH` (모바일/데스크톱 동일 비율 렌더)
- 외부 폰트: [mulmaru](https://font.ksthink.com/)
